# Dotfiles

These are my personal dotfiles for managing my Linux environment and development setup. It includes configuration for various tools like Neovim, Bash, Hyprland and more.

## Setup

### 1. **Clone the repo** into `~/.config/`:
Clone the repository into the `.config` directory to store all your dotfiles.

```bash
git clone git@github.com:mosahel01/dotfiles.git ~/.config/

# Move .bashrc to your home directory
mv ~/.config/dotfiles/.bashrc ~/

# Apply the new configuration immediately
source ~/.bashrc
