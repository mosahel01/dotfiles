# Hello, World!

These are my dotfiles.

They exist because I got tired of configuring the same things repeatedly, <br>
so decided to automate my bad decisions.

If something breaks, open an issue. <br>
If it doesn't break, that's suspicious.

<br>

### System Installation
```bash
git clone https://github.com/mosahel01/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

### Archlinux Dependencies 
```bash
cd ~/.dotfiles
./dependencies.sh
```
